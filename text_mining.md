# 精確模式

將句子最精確的切開，適合文本分析

# 全模式

把句子中所有的可以成詞的詞語都斷出来，速度非常快。

# 搜索引擎模式

在精確模式的基礎上，對長的詞語再次切分，提高召回率，適合用於搜索引擎分詞。
